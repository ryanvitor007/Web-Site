MINHAS LINGUAGENS.

HTML 5

O HTML (Linguagem de Marcação de Hipertexto) é a base da web. Ele é usado para estruturar e exibir o conteúdo na internet, definindo o que aparecerá em uma página da web e como o conteúdo é organizado.

CSS 3

O CSS (Folhas de Estilo em Cascata) é a linguagem utilizada para definir o estilo e o layout de uma página web. Ele trabalha em conjunto com o HTML, controlando a aparência de elementos, como cores, tamanhos, fontes e posicionamento.

JavaScript

O JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Ele é responsável por funcionalidades dinâmicas, como validação de formulários, criação de animações e a interação com APIs, funcionando diretamente no navegador.
